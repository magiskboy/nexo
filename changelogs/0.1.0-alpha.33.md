# 0.1.0-alpha.33

## Features

- **MCP Support**: Added support for environment variables in MCP `stdio` server connections.
- **MCP UI**: Integrated a new "Environment Variables" editor in the MCP Server connection dialog for `stdio` servers.

## Improvements

- **MCP Error Logging**: Enhanced backend error logging for MCP connection failures, including detailed messages to `stderr` and more descriptive error feedback in the UI.
- **Internationalization**: Added Vietnamese and English translations for the new environment variable settings.

## Bug Fixes

- **MCP Compatibility**: Added fallback support for legacy MCP connections using headers as environment variables.
- **Stability**: Fixed potential memory ownership issues in MCP client service logic.
