# Nexo v0.1.0-alpha.16

## Features

- **Google Gemini Tool Calling**: Full support for tool calling with Google Gemini models, including streaming and non-streaming responses.
- **Configurable Agent Iterations**: Users can now configure the `Max Agent Iterations` from the Workspace Settings (Advanced section), allowing for more control over complex task execution.

## Improvements

- **Graceful Agent Loop Termination**: When the Agent reaches the maximum number of iterations, it now provides a final summary of its findings instead of stopping with an error.
- **Improved LLM Error Handling**: Better parsing and error reporting for Google Gemini-specific API responses.

## UI/UX

- Added a new configuration field in Workspace Settings for `Max Agent Iterations`.
- Enhanced tool calling visibility for Google Gemini.
