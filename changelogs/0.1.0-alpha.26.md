# Changelog 0.1.0-alpha.26

## Features

- **Media Support**: Implement image preview modal for chat input and message attachments
- **Message Editing**: Support editing and resending messages with attached media
- **File Upload**: Integrate OpenAI file upload mechanism for non-image files
- **MCP Redesign**: Apply modern card-based grid layout to MCP server connections list

## Improvements

- **UI Refinement**: Comprehensive spacing and layout adjustments for a more compact, minimalist aesthetic
- **Image Handling**: Optimize local image loading and preview using blob URLs and efficient cleanup
- **Layout Logic**: Improve scroll management and container containment for a native app experience
- **Tooltips**: Standardize tooltip component usage across the application

## Fixes

- **LLM Integration**: Fix base64 image processing for Google and OpenAI providers
- **Memory Safety**: Resolve potential memory leaks related to image blob URLs
- **UI Components**: Fix toast animation, positioning, and layout shifts in prompt management
- **Type Safety**: Address various TypeScript and linting errors in UI components
